## Overview

DevArt Article Tools is a lightweight Joomla 6 content plugin that enhances article workflows without requiring template overrides.

The plugin provides controlled content insertion points and article enhancements while remaining compatible with modern Joomla caching and high-performance environments.

Built specifically for editorial websites where reliability, performance and maintainability are essential.

## Installation

1. Download the latest release ZIP
2. Open Joomla Administrator

`System → Extensions → Install`

3. Upload:

`plg_content_devartarticletools_v1.0.3.zip`

4. Enable:

`System → Plugins → DevArt Article Tools`

## Joomla Native Updates

DevArt Article Tools supports Joomla native updates via GitHub.

Update server:

`https://raw.githubusercontent.com/devartgr/joomla-devart-articletools/main/update.xml`

After installation:

`System → Extensions → Update`
